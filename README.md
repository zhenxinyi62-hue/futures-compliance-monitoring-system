# 期货合规监测系统

🚀 实时监测各大期货交易所违规情况和通知的网页系统。

## 📋 功能特性

- ✅ **多交易所支持** - 覆盖上期所、大商所、郑商所、中金所、广期所、能源中心
- ✅ **实时数据更新** - 自动定时爬取最新违规信息
- ✅ **强大搜索功能** - 支持按企业名称、违规内容搜索
- ✅ **灵活过滤** - 按交易所、类型、时间范围过滤
- ✅ **统计分析** - 展示总记录数、违规企业数、涉及交易所数等
- ✅ **响应式设计** - 完美适配桌面、平板、手机
- ✅ **分页显示** - 每页显示10条记录，支持分页浏览

## 🚀 快速开始

### 本地使用

1. **直接打开 HTML**
   - 在浏览器中打开 `index.html` 文件

2. **使用本地服务器**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```

3. **在线预览**
   - 通过 GitHub Pages 访问

## 📊 数据格式

```json
{
  "violations": [

      "type": "违规|通知|处罚",

    }
  ]

```

## 🔍 使用指南

### 搜索功能
- 在搜索框输入企业名称或违规内容
- 实时过滤显示相关记录

### 过滤功能
- **交易所** - 选择特定交易所查看
- **类型** - 筛选违规、通知或处罚
- **时间范围** - 查看最近7天、30天或90天的记录

### 分页
- 每页显示10条记录
- 支持上一页、下一页操作

## 💡 功能演示

1. **搜索** - 输入"中国国际"查找相关企业
2. **过滤** - 选择"上期所"查看该交易所的违规信息
3. **时间筛选** - 选择"最近30天"查看最近违规
4. **统计** - 实时显示总违规企业数、涉及交易所数
